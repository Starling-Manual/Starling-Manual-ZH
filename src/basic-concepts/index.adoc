= 基本概念

Starling可以说是一个紧凑的框架，但它仍然包含大量的包和类.
它是围绕以下几个基本概念构建的，它们旨在互补和利于扩展.
他们组织在一起共同为您提供一套工具，使您能够创建你能想象的任何应用程序.

显示编程:: 在屏幕上呈现的每个对象都是显示对象，被组织在显示列表中.
纹理和图像:: 为了将像素，表单和颜色带到屏幕，你将学习使用纹理和图像类.
动态文本:: 动态文本的渲染几乎是每个应用程序的基本任务.
事件处理:: 通信是关键！ 你的显示对象需要彼此通信，他们可以通过Starling强大的事件系统来做到这一点.
动画:: 给图形带来一些缓动！ 有不同的策略来缓动你的显示对象.
资源管理:: 了解如何加载和组织各种资源（如纹理和声音）.
特殊效果:: 效果和过滤器，将使你的图形质量脱颖而出.
实用工具:: 一些帮助工具类，使你在创作程序时更轻松.

我们的征途是星辰大海! 所以，用超级马里奥的话说: "Lets-a-go!"
