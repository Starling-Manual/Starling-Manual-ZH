本手册旨在广泛介绍 http://www.starling-framework.org[Starling 框架].
Starling是一个可用于各种应用程序, 特别关注2D游戏的ActionScript 3跨平台引擎.

ifdef::backend-html5[]
TIP: 免费下载此文档的 link:manual.pdf[PDF 版本].
endif::[]

在本手册中你将学习:

* Starling使用的技术及其遵循的原则.
* 如何选择IDE并建立第一个项目.
* 显示列表，事件和动画系统等基本概念.
* 如何挖掘片段和顶点程序的潜力等进阶技术.
* 如何发挥框架的最佳性能.
* 如何将游戏运行在手机和平板电脑上.

继续深入阅读!
